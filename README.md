# Wordle Clone

A Wordle clone built using JavaFX. The dictionary of words 
contains over 5000 English words. Application allows for restart once
correct word has been achieved.

## Usage Mac os

```
./gradlew run
```

## Usage Windows

```
gradlew run
```
